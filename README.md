# ☕ تكة - نظام إدارة المطاعم

نظام إدارة مطاعم حديث وسهل الاستخدام مبني بـ **React + Vite**

---

## 🌐 الموقع الحي
👉 **[زيارة التطبيق](https://takka-restaurant.vercel.app/)** ⚡

---

## ✨ الميزات الرئيسية

- ✅ **نظام تسجيل دخول آمن** - ثلاث صلاحيات مختلفة
  - 👔 المدير العام (Admin)
  - 👨‍🍳 نادل الصالة (Waiter)
  - 💰 محاسب الكاشير (Cashier)

- ✅ **إدارة الطاولات والطلبات** - تتبع الطلبات في الوقت الفعلي
- ✅ **نظام المحاسبة** - حساب الفاتورة والدفع
- ✅ **إشعارات فورية** - نظام تنبيهات توست
- ✅ **تزامن متعدد الأجهزة** - بيانات محدثة على جميع الأجهزة
- ✅ **واجهة عربية** - دعم كامل للغة العربية

---

## 🛠️ التكنولوجيا المستخدمة

- **React 19** - مكتبة واجهات المستخدم
- **Vite 8** - أداة البناء السريعة
- **Lucide React** - أيقونات حديثة
- **LocalStorage** - تخزين البيانات محلياً

---

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 16+
- npm أو yarn

### خطوات التشغيل

1️⃣ **استنساخ المشروع:**
```bash
git clone https://github.com/OSAMAALHALLQ1/takka.git
cd takka
```

2️⃣ **تثبيت المكتبات:**
```bash
npm install
```

3️⃣ **تشغيل التطبيق:**
```bash
npm run dev
```

4️⃣ **فتح في المتصفح:**
```
http://localhost:5173
```

---

## 📦 البناء للإنتاج

```bash
npm run build
npm run preview
```

## 👥 الأدوار والصلاحيات

| الدور | الصلاحيات |
|------|----------|
| **المدير العام** | عرض التقارير، إدارة الموظفين، المراقبة |
| **نادل الصالة** | تسجيل الطلبات، تحديث حالة الطاولات |
| **محاسب الكاشير** | حساب الفواتير، معالجة الدفع |

---

## 📝 ملاحظات

- البيانات تُحفظ في LocalStorage
- يدعم تسجيل دخول سريع عبر رابط مباشر: `?code=W-1234`
- يتم مزامنة البيانات بين علامات التبويب تلقائياً
